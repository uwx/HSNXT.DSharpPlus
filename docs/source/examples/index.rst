Examples
========

Here you will find some examples that will help you get started with DSharpPlus.

Make sure to check out our GitHub-hosted examples as well:

* C# and VB.NET: [Example bots by Emzi0767](https://github.com/Emzi0767/DSharpPlus-Example-Bot). These are fairly simple. These show the project setup and simple operations.
* C#: [Example bot by Naamloos](https://github.com/NaamloosDT/DSharpPlus-Example). This example shows some more complex features of the library.
