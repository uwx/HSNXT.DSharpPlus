    Class DspExtended

    An extension for DSharpPlus adding a variety of extension methods, and more. Some features include, but are not limited to:

    • Extension methods for DiscordChannel, DiscordMember and DiscordRole
    • Module system that works on top of DSharpPlus' extension system. See the documentation on ModuleLoader to get started!
    • A MentionReceived event
    The full version includes some more contested goodies:
    • Async listener methods registered through AsyncListenerAttribute
    • Shortcuts for modifying specific members of entities with HSNXT.DSharpPlus.Extended.ExtensionMethods.ModifyShortcuts

    Examples

    The extension methods do not require any extra setup - simply import them and use away. To install the main extension, call DspExtendedLoader.UseDspExtended(DiscordClient) or UseDspExtendedAsync(DiscordShardedClient). You can use the methods on the resulting instance to customize the fucntionality. If you're using any of the CommandsNext-related extensions, always register D#+ Extended after CommandsNext, or you will encounter crashes.

    RegisterAssembly(Assembly)

    Registers async listeners for every type (with matching method attributes) in an assembly.

    Declaration
    public void RegisterAssembly(Assembly assembly = null)
    Parameters
    Type Name Description
    Assembly assembly

    The assembly to register. If not specified, defaults to the assembly containing the current program's entry point.
    Remarks

    This method should be used with care and is usually not recommended as it performs an iteration through every single type in the assembly. Consider an overload that accepts an individual type such as Register(Type) or Register<T>().
