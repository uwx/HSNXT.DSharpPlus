    Namespace HSNXT.DSharpPlus.Extended

    Classes

    ClientModule

    A base type for modules without a configuration parameter. Children of this class must provide a constructor taking a single DiscordClient parameter that calls the base constructor with the same signature. Different or missing constructors will cause a runtime error.

    ClientModule<TConfiguration>

    A base type for modules with a configuration parameter. Children of this class must provide a constructor taking two parameters, a DiscordClient and one with the same type as TConfiguration, and it must call the base constructor with the same signature. Different or missing constructors will cause a runtime error.

    DspExtended

    An extension for DSharpPlus adding a variety of extension methods, and more. Some features include, but are not limited to:

    • Extension methods for DiscordChannel, DiscordMember and DiscordRole
    • Module system that works on top of DSharpPlus' extension system. See the documentation on ModuleLoader to get started!
    • A MentionReceived event
    The full version includes some more contested goodies:
    • Async listener methods registered through AsyncListenerAttribute
    • Shortcuts for modifying specific members of entities with HSNXT.DSharpPlus.Extended.ExtensionMethods.ModifyShortcuts

    DspExtendedLoader

    Extension class to install the DspExtended module.

    ExtensionErrorEventArgs

    ModuleLoader

    The main class for a client module. A module is just a layer of abstraction to make creating client extensions less of a burden, as it gives you access to the client in the constructor rather than an initialization method.

    ReflectionUtils

    Utility methods to reflectively access private members in DSharpPlus. Warning: all of this class' members are likely to break at some point or another, and may be changed without warning or any deprecation procedure.

    Watchdog

    A watchdog timer to detect deadlocks in threads.
