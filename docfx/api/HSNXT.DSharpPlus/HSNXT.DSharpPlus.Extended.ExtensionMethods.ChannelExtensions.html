    SyncPermissionsWithParent(DiscordChannel, DiscordChannel, String)

    Synchronizes the current channel's permissions to match the parent category's. This operation is analogous to the equivalent permission setting in the user client.

    Declaration
    public static Task SyncPermissionsWithParent(this DiscordChannel channel, DiscordChannel parent = null, string reason = null)
    Parameters
    Type Name Description
    DiscordChannel channel

    The current channel
    DiscordChannel parent

    Optional, the channel which the current channel's permissions will be synchronized to. By default, this is the current channel's parent channel category.
    String reason

    Reason string for display in audit logs. Defaults to having no reason string. The reason string will have a suffix appended to it containing the index of the current part of the operation and the total amount of parts in the operation.
    Returns
    Type Description
    Task

    A task that completes when the operation has finished.
    Remarks

    You must provide either a channel that has a parent category or the parent argument. Depending on the amount of overwrites of the parent channel, this operation may take a very long time. This method was contributed by Glockness.
