Detailed Description

Represents a generic pseudo-event listener for AwaiterHolder<TMachine,TEventArgs,TContextResult>.

It registers itself to the awaiter holder on execution, then is automatically unregistered on completion or timeout. Completion happens when the implementation of CheckResult returns a non-null value.

On completion, the underlying TaskCompletionSource<TResult> is transitioned into the "completed" state, and the awaiter is considered to have finished execution.
