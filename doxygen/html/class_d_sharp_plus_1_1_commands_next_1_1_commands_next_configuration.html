◆ EnableDefaultHelp

bool DSharpPlus.CommandsNext.CommandsNextConfiguration.EnableDefaultHelp = true
getset

Sets whether to enable default help command.

Disabling this will allow you to make your own help command.

Modifying default help can be achieved via custom help formatters (see BaseHelpFormatter and CommandsNextExtension.SetHelpFormatter<T>() for more details). It is recommended to use help formatter instead of disabling help.

Defaults to true.
