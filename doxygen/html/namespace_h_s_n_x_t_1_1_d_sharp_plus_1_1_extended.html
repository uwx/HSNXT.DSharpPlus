enum HSNXT.DSharpPlus.Extended.LogSource
strong

A type of console message in D#+ Extended. This can be used to control what gets logged by the library through.

Enumerator
None 

Don't log anything. This only affects log messages generated by D#+ Extended.

All 

Log everything from every category. Not recommended unless you want your console to get spammed.

Default 

Use the default logging settings. This is implementation-defined, but usually will log everything that isn't debug info.

EventError 

Logged when an error is encountered in a D#+ Extended event handler. Does not affect base D#+'s events, and does not stop DspExtended.ExtensionErrored from being called.

TimeoutExceeded 

Logged when a D#+ Extended event handler takes over 10 seconds to complete. Does not apply to async listener methods.

EventInvoke 

Logged when a D#+ Extended event is invoked. Does not apply to events from base D#+, or to async listener methods.

AsyncEventInvoke 

Logged when an async listener method (one that is tagged with AsyncListenerAttribute and has been registered through DspExtended.Register/RegisterAssembly) is invoked.

AsyncEventError 

Logged when an error is encountered in an async listener method (one that is tagged with AsyncListenerAttribute and has been registered through DspExtended.Register/RegisterAssembly).

WatchdogDebug 

Used for fine debug logging for the Watchdog. This usually does not need to be enabled unless you are testing the watchdog.

ExtensionDebug 

Used for fine debug logging for the extension. Usually only needed for testing the extension itself.
