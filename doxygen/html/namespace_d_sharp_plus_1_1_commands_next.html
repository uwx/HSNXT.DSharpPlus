delegate Task<int> DSharpPlus.CommandsNext.PrefixResolverDelegate ( DiscordMessage  msg)

Represents a delegate for a function that takes a message, and returns the position of the start of command invocation in the message. It has to return -1 if prefix is not present.

It is recommended that helper methods CommandsNextUtilities.GetStringPrefixLength(DiscordMessage, string, StringComparison) and CommandsNextUtilities.GetMentionPrefixLength(DiscordMessage, DiscordUser) be used internally for checking. Their output can be passed through.

Parameters
msgMessage to check for prefix.
Returns
Position of the command invocation or -1 if not present.
