)

Sends a message with a "page", with reactions to move between pages. Task resolves when ct is cancelled, timeout is hit, or PaginationEmojis.Stop is clicked.

Parameters
channelThe channel to send the paginated message in
userThe user that can interact with the pages
messagePagesAny amount of Pages to use
ctCancellation token to skip the timeout and instantly stop the pagination
timeoutTimeout for the waiting period. If not specified, defaults to InteractivityConfiguration.Timeout. The timeout is reset whenever an action is taken (e.g clicking a button)
timeoutBehaviourOverrideBehavior for when the pagination finishes (e.g timeout is hit). Defaults to InteractivityConfiguration.PaginationBehavior.
emojisSet of PaginationEmojis to use for controlling the pages. Defaults to InteractivityConfiguration.DefaultPaginationEmojis.
Returns
Task that resolves when pagination ends
Exceptions
ArgumentNullExceptionIf channel , user or messagePages is null
ArgumentExceptionIf there are no pages to send
